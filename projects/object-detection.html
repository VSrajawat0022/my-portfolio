Cricbuzz Clone

A real-time cricket score update system using the MERN stack.

Project Overview

Cricbuzz Clone is a web application that provides real-time cricket score updates. It is built using the MERN stack with WebSockets for live updates.

Tech Stack

  • MongoDB
  • Express.js
  • React.js
  • Node.js
  • WebSockets

Challenges Faced

Handling real-time updates efficiently was challenging. Using WebSockets helped reduce server load and improve performance.

My Contributions

  • Developed the backend API for fetching live scores
  • Implemented WebSockets for real-time updates
  • Designed the UI with React and Tailwind CSS
